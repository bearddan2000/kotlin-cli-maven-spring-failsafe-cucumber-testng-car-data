# kotlin-cli-maven-spring-failsafe-cucumber-testng-car-data

## Description
A POC for spring app using testng
and cucumber framework with failsafe
and surefire plugins.

## Tech stack
- kotlin
- maven
  - spring
  - testng
  - cucumber
  - failsafe

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
